# CSF_lab03_02250340
# CSF Lab 03

## Aim of the Lab
To develop a clear understanding of control structures in Python by implementing conditional statements, loops, break and continue statements, and functions. This lab also aims to improve problem-solving skills by writing structured and efficient programs.

---

## List of Programs
1. conditional statement
2. loops
3. break and continue
4. functions
5. integrated program 

### Part A: Conditional Statements
1. Check whether a number is positive, negative, or zero  
2. Check whether a number is even or odd  
3. Find the largest of two numbers  
4. Find the largest of three numbers  
5. Display grade based on marks  
6. Menu-driven calculator using if-elif  

### Part B: Loops
7. Print numbers from 1 to 10  
8. Print even numbers from 1 to 20  
9. Find the sum of first 10 natural numbers  
10. Display multiplication table  
11. Print numbers using while loop  
12. Calculate sum from 1 to n using while loop  
13. Take input until user enters 0  

### Part C: Break and Continue
14. Print numbers using break  
15. Print numbers skipping a value using continue  
16. Search for a number in a list  

### Part D: Functions
17. Print “Hello, World!” using function  
18. Find square of a number  
19. Add two numbers using function  
20. Check even or odd using function  
21. Calculate factorial  
22. Print multiplication table using function  

### Part E: Integrated Programs
23. Print numbers from 1 to n and check even/odd  
24. Menu-driven calculator using functions and loop  

---

## Brief Explanation of Each Task

1. Determines whether a number is positive, negative, or zero using if-else.  
2. Checks if a number is even or odd using modulus operator.  
3. Compares two numbers and displays the larger one.  
4. Uses nested if-else to find the largest among three numbers.  
5. Assigns grades based on marks using conditional statements.  
6. Performs basic arithmetic operations using a menu-driven approach.  

7. Uses a for loop to print numbers from 1 to 10.  
8. Prints all even numbers within a given range.  
9. Calculates the sum of the first 10 natural numbers.  
10. Generates multiplication table of a given number.  
11. Demonstrates while loop by printing numbers from 1 to 5.  
12. Calculates sum from 1 to n using a while loop.  
13. Continuously takes input until 0 is entered.  

14. Demonstrates use of break to exit loop early.  
15. Demonstrates use of continue to skip an iteration.  
16. Searches for a number in a list and stops when found.  

17. Defines and calls a simple function.  
18. Returns square of a number using a function.  
19. Adds two numbers using a function.  
20. Checks even or odd using a function.  
21. Calculates factorial using a function.  
22. Prints multiplication table using a function.  

23. Uses function and loop to check even/odd for numbers from 1 to n.  
24. Implements a menu-driven calculator using functions and loop until exit.
